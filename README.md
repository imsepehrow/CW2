# CW2

## Project Description
This project was developed for the client so the client could managed their stock effeciently, and effectively using a command-line application.  

## This project allows the client to:
- Add stock to the inventory
- View the current stock in a structured table
- Update and Delete stock from the inventory
- Search for stock within the inventory
- Veiw which items are low in stock
- Automatically save changes made

## Insallation and Usage
1. Clone the repository
2. Open main.py file
3. Run main.py
